image uploader app

A small FullStack MERN typescript app where you can upload your images to amazon web services (AWS S3). The back-end part of the application I used Multer a node.js middleware primarily used to upload files. @aws-sdk/client-s3 AWS SDK for Javascript S3 Client. @aws-sdk/s3-request-presigner This package generate presigned url from S3 client.


On the front-end part Reactjs and Tailwindcss to build the app interface. React Dropzone simple React hook to create drag'n'drop zone. And Axios to make http request for the browser and nodejs.
